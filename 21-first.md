###  React 牵手 Express

    前端和后端是完全独立的两个项目，通过 API 来作为桥梁。其实，我们后端虽然是 Nodejs 写的，但是用 PHP/JAVA 也能写。

    react搭建环境(回顾)

    ```

    mkdir suiliyan441
    cd suiliyan441
    atom .
    ```

    生成package.json

    ```

    npm init

    ```

    安装四个包

    ```

    npm i --save weebpack babel-core babel-preset-env babel-loader

    ```
